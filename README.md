# 🫧 비눗방울 페르마 점 시뮬레이션

3D 다면체를 비눗방울 용액에 담가 페르마 점을 관찰할 수 있는 인터랙티브 시뮬레이션입니다.

## 🚀 실행 방법

### 1. 필요 프로그램 설치
먼저 Node.js가 설치되어 있어야 합니다.
- [Node.js 다운로드](https://nodejs.org/) (v18 이상 권장)

### 2. 패키지 설치
터미널에서 프로젝트 폴더로 이동한 후 다음 명령어를 실행하세요:

```bash
npm install
```

### 3. 개발 서버 실행
```bash
npm run dev
```

실행 후 브라우저에서 `http://localhost:5173` 으로 접속하세요.

### 4. 빌드 (배포용)
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 🎮 사용 방법

1. **다면체 선택**: 정사면체, 정육면체, 정팔면체 중 선택
2. **담그기**: "비눗방울 용액에 담그기" 버튼 클릭
3. **페르마 점 관찰**: 중심점에서 각 꼭짓점으로 연결된 최단 경로 확인
4. **빨대 조작**: 
   - 빨대를 드래그하여 이동
   - 빨대를 클릭하여 비눗방울 불기
5. **표면 막 관찰**: 페르마 구조가 사라지고 표면 막만 남음

## 🖱️ 마우스 컨트롤

- **드래그**: 다면체 회전
- **빨대 드래그**: 빨대 이동 (빨대가 보일 때)
- **빨대 클릭**: 비눗방울 불기 (페르마 구조 상태일 때)

## 🛠️ 기술 스택

- React 18
- Three.js
- Tailwind CSS
- Vite

## 📚 페르마 점이란?

여러 점에서 최단 거리로 연결되는 점을 페르마 점이라고 합니다. 
비눗방울이 다면체 내부에 형성될 때, 자연스럽게 표면 에너지를 최소화하려는 
성질 때문에 페르마 점 구조를 만들게 됩니다.
